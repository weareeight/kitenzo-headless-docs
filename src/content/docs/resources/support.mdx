## What to include

To help us help you quickly:

- Your **store domain** (`your-store.myshopify.com`).
- The **bundle ID(s)** involved, if relevant.
- Your **integration mode** (Embed or SDK) and framework (Hydrogen, Next.js, custom).
- For API issues: the **endpoint**, request, and the **status code** / error body you got back.

## Common requests

| I want to… | Do this |
|---|---|
| Enable Headless on my store | Email [support@kitenzo.com](mailto:support@kitenzo.com) — see [Get access](/getting-started/access/). |
| Raise my rate limit | Email support with your key and expected traffic. |
| Get a test (`kit_test_`) key | Ask support. |
| Report a docs/data inaccuracy | Email support; the API and published SDK are the source of truth. |
